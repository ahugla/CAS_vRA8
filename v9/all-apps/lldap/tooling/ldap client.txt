ldapsearch -x -b "dc=myfrlab,dc=com" -H ldap://10.1.11.8:3890  -D "cn=admin,ou=people,dc=myfrlab,dc=com" -W     ( “-W” in order to be prompted for the password)

  -x : Trust any certificate 
  -b : Specifies the base DN that should be used for the search.
  -H : directory server
  -D : bindDN, The DN to use to bind to the directory server when performing simple authentication
  -W : in order to be prompted for the password
